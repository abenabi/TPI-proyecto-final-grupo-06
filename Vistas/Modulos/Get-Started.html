Iniciando C++

Para iniciar en C++ necesitas

  • Un editor de texto como: DEV C++, Visual studio code, notepad++, bloc de notas, entre otros
  • Un compilador

El programa Dev C++ es una gran herramienta ya que es un editor de texto y al mismo tiempo un compilador, adjunto el link de Dev C++

https://sourceforge.net/projects/orwelldevcpp/

Tambien recomiendo usar Visual Studio Code, es una editor de texto con muchos pluggins y extensiones que son de mucha ayuda, adjunto link de descarga de Visual Studio Code y un tutorial de como instalarlo

https://code.visualstudio.com/
